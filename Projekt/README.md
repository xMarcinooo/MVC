# Film Catalog

Prosta aplikacja ASP.NET Core MVC do zarządzania katalogiem filmów.

## Spis treści

- Opis projektu
- Funkcjonalności
- Uruchomienie

## Opis projektu

Aplikacja umożliwia przechowywanie informacji o filmach i zarządzanie kolekcją filmową.

## Funkcjonalności

- wyświetlanie listy filmów,
- dodawanie nowych filmów,
- edycja istniejących filmów,
- usuwanie filmów,
- podgląd szczegółów filmu.

Dla każdego filmu przechowywane są:
- tytuł,
- reżyser,
- ocena.

## Uruchomienie

1. Sklonuj repozytorium:
   ```bash
   git clone [adres_repozytorium]
   ```

2. Otwórz projekt w Visual Studio.

3. W Konsoli Menedżera Pakietów wykonaj:
   ```powershell
   Update-Database
   ```

4. Uruchom aplikację klawiszem F5.

## Technologie

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server 

## Wymagane pakiety

Projekt korzysta z:

- Microsoft.EntityFrameworkCore
- Microsoft.EntityFrameworkCore.SqlServer
- Microsoft.EntityFrameworkCore.Tools
